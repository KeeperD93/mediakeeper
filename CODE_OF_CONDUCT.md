# Code of Conduct

MediaKeeper adopts the **Contributor Covenant, version 2.1** as its code of
conduct for all interactions in this repository (issues, pull requests,
discussions, code reviews and any other contribution channel).

The full text of the Contributor Covenant 2.1 is the canonical reference:

- HTML version: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Markdown version: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

Contributors and maintainers are expected to read it once and apply it in
every interaction with the project.

## Reporting a concern

If you witness or experience behaviour that violates the Contributor
Covenant in this project's spaces, please report it privately:

1. Preferred: open a **private GitHub Discussion** (if Discussions are
   enabled on the repository) or contact the maintainer through their
   GitHub profile.
2. Fallback: open a brief, non-revealing public issue asking for a
   private channel and a maintainer will reply with one. **Do not include
   the details of the incident in that initial public issue.**

All reports will be handled confidentially.

## Enforcement

The maintainers are responsible for clarifying and enforcing the standards
of acceptable behaviour and will take appropriate and fair corrective
action in response to any behaviour they deem inappropriate, threatening,
offensive or harmful.

The enforcement guidelines defined in the Contributor Covenant 2.1 (see
the full text linked above) describe the response a contributor can
expect depending on the severity of the violation.

## Scope

This Code of Conduct applies within all project spaces (this GitHub
repository, its issues, pull requests, discussions and any other
official communication channel) and also when an individual is
officially representing the project in public spaces.
