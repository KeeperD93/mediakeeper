# Code de conduite

<p align="center">
  <a href="CODE_OF_CONDUCT.md">English</a> · <b>Français</b>
</p>

MediaKeeper adopte le **Contributor Covenant, version 2.1** comme code de conduite pour toutes les interactions dans les espaces du projet (issues, pull requests, discussions, revues de code, Discord, tout autre canal de contribution).

Le texte complet fait foi :

- HTML : <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Markdown : <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>
- Traduction française non officielle : <https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/>

Les contributeurs et mainteneurs sont attendus à le lire une fois et à l'appliquer dans chaque interaction.

---

## Signaler une situation

Si vous êtes témoin ou victime d'un comportement contraire au Contributor Covenant dans les espaces du projet, merci de le signaler **en privé** :

1. **Préféré** — message direct à un mainteneur sur le serveur Discord du projet (`<DISCORD_INVITE>`).
2. **Alternatif** — ouvrir une issue publique brève et non révélatrice demandant un canal privé ; un mainteneur vous répondra avec un canal. **N'incluez pas les détails de l'incident dans cette issue publique initiale.**

Tous les signalements sont traités de manière confidentielle.

---

## Application

Les mainteneurs sont responsables de clarifier et de faire appliquer les standards de comportement acceptables. Ils prendront des mesures correctives appropriées et équitables face à tout comportement qu'ils jugent inapproprié, menaçant, offensant ou nuisible.

Les lignes directrices d'application définies dans le Contributor Covenant 2.1 (voir le texte complet lié plus haut) décrivent la réponse à laquelle un contributeur peut s'attendre selon la sévérité de la violation.

---

## Périmètre

Ce code de conduite s'applique dans tous les espaces du projet (ce dépôt GitHub, ses issues, pull requests, discussions, le serveur Discord, et tout autre canal de communication officiel) ainsi que lorsqu'une personne représente officiellement le projet dans des espaces publics.
